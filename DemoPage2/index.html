Jungle

The word jungle originates from the Sanskrit word jaṅgala (जङ्गल), meaning rough and arid. It came into the English language in the 18th century via the Hindustani word for forest (Hindi/Urdu: जङ्गल/جنگل) (Jangal). Jāṅgala has also been variously transcribed in English as jangal, jangla, jungal, and juṅgala.[citation needed] It has been suggested that an Anglo-Indian interpretation led to its connotation as a dense "tangled thicket".

Eiffel Tower

The Eiffel Tower (/ˈaɪfəl/ ⓘ EYE-fəl; French: Tour Eiffel [tuʁ ɛfɛl] ⓘ) is a wrought-iron lattice tower on the Champ de Mars in Paris, France. FAQs

What is difference between inline-block and block Elements?

Block elements take up the full width of their container and always start on a new line (e.g., div, p ). Inline-block elements behave like inline elements but allow setting width and height while staying in line with other elements (e.g., span, img, button).

What is difference between resert and normalize css?

Reset CSS removes all default browser styles, making elements appear the same across different browsers but requiring more custom styling. Normalize CSS preserves useful default styles while fixing inconsistencies, ensuring better cross-browser consistency with minimal overrides.

What is font face and why is it used?

@font-face is a CSS rule that allows custom fonts to be loaded on a webpage instead of relying on system fonts. It is used to enhance typography, maintain brand identity, and improve design consistency across different devices and browsers.

which are the most aplicable font-face type for all browsers?

The most widely supported @font-face formats for all browsers are WOFF, WOFF2, TTF, and EOT. WOFF (Web Open Font Format) is optimized for web use and works across modern browsers, while WOFF2 offers better compression and faster loading times. TTF (TrueType Font) is broadly supported but less optimized for the web. EOT (Embedded OpenType) is mainly required for older versions of Internet Explorer (IE 6-8). Although SVG fonts were once used for older iOS browsers, they are now mostly obsolete. For the best compatibility, it's recommended to include WOFF, WOFF2, and TTF in your @font-face rule.

What is the difference between CSS Grid and Flexbox?

CSS Grid and Flexbox are both layout models in CSS that allow for the creation of complex layouts with ease. CSS Grid is a two-dimensional layout system that allows for the creation of grid-based layouts with rows and columns. Flexbox is a one-dimensional layout system that allows for the creation of flexible layouts with rows or columns. CSS Grid is best suited for creating complex layouts with multiple rows and columns, while Flexbox is best suited for creating simple layouts with rows or columns. CSS Grid is more powerful and flexible than Flexbox, but Flexbox is easier to use and more widely supported.
